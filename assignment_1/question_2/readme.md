# Frailty Dataset Analysis
Principles of Data Science Assignment 1 Question 2

## Project Overview
This project provides exploratory data analysis on the StudentsPerformance Dataset.

## Dataset
The raw dataset contains the following:
- gender: (categorical) `'female'`, `'male'`
- race/ethnicity: (categorical) `'group B'`, `'group C'`, `'group A'`, `'group D'`, `'group E'`
- parental level of education: (categorical) `'bachelor's degree'`, `'some college'`,    `'master's degree'`, `'associate's degree'`, `'high school'`, `'some high school'`
- lunch: (categorical) `'standard'`, `'free/reduced'`
- test preparation course: (categorical) `'none'`, `'completed'`
- math score: (integer 0-100)
- reading score: (integer 0-100)
- writing score: (integer 0-100)
